def gcj2towgs(lat,lon):
lat=float(lat)
lon=float(lon)
a = 6378245.0 # 克拉索夫斯基椭球参数长半轴a
ee = 0.00669342162296594323 #克拉索夫斯基椭球参数第一偏心率平方
PI = 3.14159265358979324 # 圆周率
# 以下为转换公式
x = lon - 105.0
y = lat - 35.0
# 经度
dLon = 300.0 + x + 2.0 * y + 0.1 * x * x + 0.1 * x * y + 0.1 * math.sqrt(abs(x))
dLon += (20.0 * math.sin(6.0 * x * PI) + 20.0 * math.sin(2.0 * x * PI)) * 2.0 / 3.0
dLon += (20.0 * math.sin(x * PI) + 40.0 * math.sin(x / 3.0 * PI)) * 2.0 / 3.0
dLon += (150.0 * math.sin(x / 12.0 * PI) + 300.0 * math.sin(x / 30.0 * PI)) * 2.0 / 3.0
#纬度
dLat = -100.0 + 2.0 * x + 3.0 * y + 0.2 * y * y + 0.1 * x * y + 0.2 * math.sqrt(abs(x))
dLat += (20.0 * math.sin(6.0 * x * PI) + 20.0 * math.sin(2.0 * x * PI)) * 2.0 / 3.0
dLat += (20.0 * math.sin(y * PI) + 40.0 * math.sin(y / 3.0 * PI)) * 2.0 / 3.0
dLat += (160.0 * math.sin(y / 12.0 * PI) + 320 * math.sin(y * PI / 30.0)) * 2.0 / 3.0
radLat = lat / 180.0 * PI
magic = math.sin(radLat)
magic = 1 - ee * magic * magic
sqrtMagic = math.sqrt(magic)
dLat = (dLat * 180.0) / ((a * (1 - ee)) / (magic * sqrtMagic) * PI)
dLon = (dLon * 180.0) / (a / sqrtMagic * math.cos(radLat) * PI)
wgsLon = lon - dLon
wgsLat = lat - dLat
return {‘lon’:wgsLon,’lat’:wgsLat}

随机森林

2020-05-28

目前的集成学习算法主要有两大派系，一个是boosting算法为代表的强依赖关系个体学习器，必须串行生成的序列化方法；另一个是以Bagging和“随机森林”为代表的算法，其个体学习器间不存在强依赖关系，并能够同事生成的并行化方法。

​ 结合策略有：平均法、投票法和学习法等。平均法常用语数值类型的回归预测，对于若干个弱分类器进行算术平均或加权平均得到最终的预测输出。投票法是主要针对分类问题的预测，它通过对不同弱分类器结果进行投票，少数服从多数。学习法则是对弱分类器结果进行处理再加上一层学习器，重新训练一个分类器来得到最终结果。

Boosting算法

​ 主要专注于提高弱分类算法的精确度方面。

​ 方法：将样本集分割成若干个样本子集，每一个样本集赋予相同权重，给基分类器进行分类，分类错误的分类器的样本集提高权重（实际是正确的降低权重），然后训练第二个训练集，以此类推，产生n个分类器，最后Boosting框架算法将这n个基分类器进行加权合并，产生一个最后的结果分类器。Adaboost就是Boosting中较为代表的算法。

Bagging方法

​ 随机有放回得选择训练数据，在此基础上构造分类器，然后组合到一起。每一次随机采样生成一个采样集，也能独立训练一个弱分类器，如此进行n次，就得到n个弱分类器，再进行组合成最终的强分类器。

随机森林

随机森林是以决策树这种弱分类器为基础，进行进阶的一种Bagging集成学习算法。在Bagging随机采样的基础上，又加上了特征的随机选择。

步骤：

（1） 重抽样生成k个样本子集，理论上覆盖2/3的数据，未包含的数据作为测试数据。

（2） 利用k个样本子集生成k个决策树，每棵树从M个特征中随机选择m个特征（实际项目中通常采用m^1/2)，充分生长不做剪枝。

（3） 综合每棵树的测试结果按照一定的投票机制确定最终结果。

投票机制有很多，除了常见的少数服从多数还有一票否决法，阈值法与贝叶斯投票法。

一票否决法就是当且仅当所有决策树都对此实例进行如此分类才可以，否则拒绝。

阈值法则是统计实例分为正样本和负样本的数目，如果达到事先定好的阈值则划分到该类上。

贝叶斯投票机制[1]就是根据不同决策树非分类效果赋予不同的权重，最后进行加权计算。

优点：随机森林算法很好的利用了随机性，最小化了各树之间的相关性，提高了分类性能。同时，森林可以实现并行化，分类速度非常快。

极端随机森林

区别：

（一） 随机森林采用随机生成的样本子集来训练决策树，而极端随机森林采用所有的训练样本得到每一棵决策树。

（二） 随机森林在随机子集中得到最佳分裂属性，而计算随机森林采用完全随机得到分裂值，再选择分类值较大的划分。

注释：

[1]贝叶斯平均法　

C是补偿评价数（平均评价数），M是补偿评分（平均评分），n是评价数量，s是已计算评分。通常C、M应为常量，使最终评分只与评价数量和已计算评分相关。

在这个公式中，M（总体平均分）是”先验概率”，每一次新的投票都是一个调整因子，使总体平均分不断向该项目的真实投票结果靠近。投票人数越多，该项目的”贝叶斯平均”就越接近算术平均，对排名的影响就越小。

决策树

2020-05-28

决策树可以解释成一种特殊形式的规则集，特征是规则的层次组织关系。只能对离散型变量进行学习。决策树是一种贪心算法。从分类原理和数据训练来看，逻辑回归、贝叶斯分类属于利用统计学原理对数据集进行分类（基于回归分析、概率论等），决策树分类采用归纳学习的方法（通过信息论的相关理论对数据差异进行归纳，建立类别标准）。

关键：如何选择合适的分裂属性进行分裂，以及如何制定合适的分裂谓词产生相应的分支。（各种决策树主要区别也在于此）

衡量标准：信息熵，通过引入信息增益概念，来表示划分数据集前后信息发生的变化

约束：任由决策树自然生长容易过拟合，增加学习错误率。简化决策树的方法有：控制树的规模、修改测试空间、修改测试属性、数据库约束、改变数据结构等。

控制数的规模：就是控制树的分支数量，一般采用预剪枝算法、后剪枝算法以及增量树方法来实现。预剪枝就是停止树的扩张，何时停止是研究的主要内容。后剪枝算法先生成一棵未剪枝的决策树，再进行可能的修建，把它在作为输入，再利用修剪集进行选择，输出最好的规则。

缺点：

1.连续性数据难以处理

2若数据存在事件顺序，需要预处理

3.特征很多的数据，由于信息量太大，反而容易出错

4.分类只是根据一个字段.

主要生成算法：ID3、C4.5、CART等（ID3、C4.5仅用于分类，CART可以用于分类和回归）。

ID3算法

采用信息增益作为属性选择标准，本质就是使用这个该属性分割会导致系统熵降低，所以信息增益越大，不确定性减少越多。

C4.5算法

C4.5算法是对ID3算法的延伸和优化，ID3采用信息增益选择分裂属性，如果存在一对一的标志属性，容易过拟合，产生“归纳偏置”。

而C4.5采用信息增益率来选择分裂属性：

​ 信息增益率为=信息增益/分裂信息

可以看出，相当于给信息增益加了一个惩罚项，避免过大或者过小的极端情况。

ID3无法处理连续型数据，而C4.5可以通过对某个连续性属性进行处理：

\1. 属性的取值从小到大排序

\2. 两个属性取值之间的重点作为可能的分裂点，将该节点的数据集分成两份，计算每个可能的分裂点的信息增益

\3. 计算每一种分割对应的信息增益率，选择信息增益率最大的进行分割。

CART算法

分类回归树CART是一种典型的二叉决策树（对特征属性进行二元分裂），主要用来研究分类问题，可以同时处理连续变量和分类变量。根据目标变量不同，可以分为分类决策树和回归决策树。

首要目标就是构造一个准确的分类模型进行预测，即研究引起分类现象发生的变量及变量之间的作用，通过建立决策树和决策规则来对未知类型的变量进行类别预测。

采用Gini系数最小化准则来进行特征选择，生成二叉树。

预剪枝

决策树生成之前进行剪枝，由于缺乏对整个分类过程、分类规则的全面了解，他是通过提前停止树的构造实现的。例如，给定节点不在分裂；指定节点熵小于某个值；此节点实例个数小于某一个阈值也可以停止树生长。

后剪枝

允许决策树过拟合训练数据，然后自底向上或自顶向下对非叶子节点进行考察，若该节点对应的子树换为叶子节点能够带来泛化性能的提升，则把该子树替换为叶子节点。常见方法有：错误率降低剪枝（REP）和悲观错误剪枝。

REP方法：

​ 数据分为验证集和训练集。从树最底部开始，自下而上，从树中每一个非叶子节点开始，进行一次剪枝，用新生成的树去训练验证集，如果与剪枝之前分类偏差相差不大或相等，并且没有相同性质的新子树生成，则剪枝成功。随着剪枝的进行，直到某一次修剪降低了决策树精度，该剪枝过程终止。

​ 当训练数据集过少时，训练集和验证集划分不同，容易导致部分数据误删，通常不考虑REP算法。

贝叶斯分类

2020-02-10

贝叶斯分类以贝叶斯定理为基础的一类分类算法的总称。朴素贝叶斯分类常常被称为生成式分类器，它是基于事物的概率特征来实现分类的。而逻辑回归常常被称为判别式分类器，其中的概率只是一种分类的判别方式。

贝叶斯定理

​ 可以粗略地简述成：若要预见未来，必须回首过往。

​ 也就是说未来某件事发生的概率可以通过计算他过去发生的频率来估计。

​ 具体参考贝叶斯定理文章。

原理

​ 贝叶斯分类模型是一种典型的基于统计方法的分类模型。贝叶斯决策系统把分类问题看做一种不确定性决策问题。其决策依据是分类错误率最小或者损失风险最小。他通过训练集训练归纳出分类器，并利用分类器对没有分类的数据进行分类。其中具有代表性的有朴素贝叶斯分类器、贝叶斯网络分类器和树扩展的朴素贝叶斯分类模型TAN分类器等。

特点

（一） 贝叶斯分类是计算出属于某一类的概率，具有最大概率的类便是该对象所属的类。

（二） 贝叶斯分类中所有的属性都潜在地起作用。

（三） 贝叶斯分类对象的属性可以是离散的、连续的也可以是混合的。

在众多分类其中朴素贝叶斯具有简单的结构和良好的性能。它是基于贝叶斯定理和特征条件独立假设的分类方法。它主要进行了两方面的假设，第一是特征出现的概率是相同的；第二是每个特征同等重要。他之所以朴素是因为它的假设条件以及原理都很朴素。

朴素贝叶斯算法是假设各个属性都是独立的，当各个属性都不独立时，那就需要采用贝叶斯网络了。

贝叶斯网络结构形式

（一）tail-to-tail

1.在c未知的时候，有：P(a,b,c)=P(c)P(a|c)P(b|c)，此时，没法得出P(a,b) = P(a)P(b)，即c未知时，a、b不独立。

2.在c已知的时候，有：P(a,b|c)=P(a,b,c)/P(c)，然后将P(a,b,c)=P(c)P(a|c)P(b|c)带入式子中，得到：P(a,b|c)=P(a,b,c)/P(c) = P(c)P(a|c)P(b|c) / P(c) = P(a|c)*P(b|c)，即c已知时，a、b独立。

（二）head-to-tail

c未知时，有：P(a,b,c)=P(a)P(c|a)P(b|c)，但无法推出P(a,b) = P(a)P(b)，即c未知时，a、b不独立。

c已知时，有：P(a,b|c)=P(a,b,c)/P(c)，可化简得到：P(a,b|c)= P(a|c)*P(b|c) 即c已知时，a、b独立。

拉格朗日对偶

2020-01-22

在学习线性可分支持向量机的最优化问题过程中，需要对其求解，此时就会用到拉格朗日对偶性的相关知识。其核心思想为，应用拉格朗日对偶性，将原始的求解最优化问题，转化为求解对偶问题的最优解，这就是线性可分支持向量机的对偶算法。

首先我们需要知道为什么要这样做？

原始问题是一个带有约束的优化问题，如果不带约束，因为原函数连续可微，我们可以直接通过求导，然后令其为零，直接就可以求出最优解。可问题是带有约束的，我们知道如果不带约束会简单很多，如何把约束去掉呢？拉格朗日函数可以做到。

任何带约束的优化问题总是可以写出拉格朗日函数来。引入拉格朗日乘子，原函数可以转化成广义的拉格朗日函数一般式子。现在问题是，该怎么把这两个式子联系起来。我们知道，一般约束可以概括成两种，一种为不等式，一种为恒等式，而且通过转化都可以写成一边为零的形式。显然由拉格朗日一般式可以看出，对于任意两种拉格朗日乘子，取到某值使得拉格朗日式子最大时，此时拉格朗日函数等于原函数。当拉格朗日乘子取得最大值时，函数变成x的函数，依照原问题，我们需要取最小值。

此时求解会很麻烦，我们需要转化一下，将原始问题转化为对应的对偶问题。

熵笔记

在了解熵的概念之前，需要先知道信息的概念。一个事件发生是有概率的，而这个事件发生的概率越大，它所蕴含的混乱度就越小，就越趋于稳定，也就是蕴含的信息量最小，相反，这个事件发生的概率越小，他就越混乱，不确定性就越多，所蕴含的信息量就越大。

在信息论和概率统计中，熵是表示随机变量不确定性的度量。也可以理解为蕴含信息量的多少，即信息的期望值。我比较倾向于以信息的概念来理解熵。我们知道概率是在0到1之间的数，概率越大越接近1，它所蕴含的信息量就是越小，概率越小越靠近0，它所蕴含的信息量就越巨大。这时候我们能想到信息函数是一个对数函数：

L(Xi)=-log2P(Xi)

P(Xi)表示第i个分类的概率函数，Xi表示第i个分类。

在随机变量X给定条件下随机变量Y的不确定性，实质上是H(X,Y)-H(X),X和Y都发生的信息量减去X确定发生的信息量，得到的就是在X确定发生情况下Y发生的信息量。容易混淆的是，此时随机变量Y的条件熵不是X给定条件下的信息熵，而是还要对其求期望。

​

信息增益表示得知特征X的信息而使得类Y的信息不确定性减少的程度。一般地，熵H(Y)与条件熵H(Y|X)之差成为互信息。而两个随便变量X和Y互信息的定义则为X，Y的联合分布和独立分布乘积的相对熵。两个随机变量函数越接近，即独立性越小，则值越大，相反，值越小说明独立性越大，等于0时，两随机变量分布函数独立分布。在决策树学习中，我们认为信息增益等价于互信息。

根据之前所学条件熵信息熵的定义，结合互信息定义，可以推导出：

I(X,Y)=H(X)+H(Y)-H(X,Y)

相对熵的概念就是可以理解为两个随机变量的距离。

随机性探讨

物理层面上出现随机性是非常少见的，物理规律通常认为是确定的。可是我们身处的社会规律，越来越多的接受了随机性的假设。在以前，统计方法都运用得很少，玻尔的老师卢瑟福曾经说过：“如果你的实验需要统计学，那么你应该再做个更好的实验。”

在理性的基础上，所有的判断都是统计学。

遗传学家霍尔丹曾说过：”人类是一种常规动物，并不能模仿自然界的无序。”生活中有各种“随机性”的错觉，事实上，人们很难抹去对随机的执念，怕比例失衡。比如投硬币多次正面后，总会不自觉的想象一些反面的情况。

贝叶斯定理可以通过反馈不断更新概率从而精准预测。不需要考虑其他方面因素，可以成为先验概率或边缘概率。

推断统计学中，我们要习惯不要说接受原假设，即使原假设概率很大，也只能用不能拒绝这样的字眼，只有备选假设才能接受。
